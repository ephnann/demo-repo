Importing Images

The image above has been imported using the CSS property "background-image: url()". This is a good method for creating responsive images such as hero images. The advantage of using CSS to import images is that you can:
1) align to grid

2) Use alignment such as 'cover' and 'center'

The downside of this method is that the size and positon of the image is less predictable.

The image above has been imported using a simple image tag. These images need to have a width and a height defined. This remains static and cannot be responsively aligned to a grid. The advantage of such images is:

1) They will not automatically clip. They are good for things like logos.
